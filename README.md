# AI 신약개발 학습 프로그램

AI + Computational Chemistry 분야에서 신약개발 파이프라인과 도구를 개발하기 위한 체계적 학습 프로그램입니다.

## 빠른 시작

```bash
pip install -r requirements.txt
python app.py
```

브라우저에서 `http://localhost:5000` 접속

## 커리큘럼 (20주 / 5 Phase)

| Phase | 주제 | 기간 |
|-------|------|------|
| 1 | ML/DL 기초 | Weeks 1-4 |
| 2 | 화학정보학 (RDKit) | Weeks 5-8 |
| 3 | AI 신약개발 | Weeks 9-12 |
| 4 | 고급 주제 (도킹/MD/ADMET) | Weeks 13-16 |
| 5 | 파이프라인 구축 | Weeks 17-20 |

## 기능

- 오늘의 학습 대시보드
- 진도 추적 및 시각화
- 인터랙티브 퀴즈
- 커리큘럼 브라우저
- Claude Code 튜터 연동
