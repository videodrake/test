# Phase 1: ML/DL 기초

## 목표
머신러닝과 딥러닝의 핵심 개념을 **분자 데이터** 맥락에서 학습합니다.

## 선수 지식
- Python 중급 (함수, 클래스, 패키지 관리)
- 기초 수학 (미적분, 행렬 연산)

## 주차별 구성
- **Week 1:** Python 과학 컴퓨팅 (NumPy, Pandas, Matplotlib)
- **Week 2:** scikit-learn으로 QSAR 모델링
- **Week 3:** PyTorch 딥러닝 기초
- **Week 4:** 분자 데이터용 고급 딥러닝 (GNN, Attention)

## 마일스톤 프로젝트
**분자 물성 예측기**: Random Forest와 GNN을 비교하여 분자 용해도를 예측하는 완전한 ML 파이프라인 구축
